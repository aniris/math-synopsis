Дифференциальные уравнения. Основные понятия

Дифференциальное уравнение (Д.У.) — это такое уравнение, которое связывает функцию (одной или нескольких переменных), аргумент (или несколько аргументов) и производные различных порядков от данной функции.
При этом самый большой порядок входящих производных называется порядком уравнения.

$ y = f(x) $ — обыкновенное уравнение одной переменной
$ y = \phi(x, y, y', y'', ... y^{(n)}) = 0 $ — общий вид ДУ
Если уравнение $ y^{(n)} $ можно представить в виде:
$ y^{(n)} = \phi(x,y,y',... y^{(n - 1)}) $ — ДУ, разрешенное относительно $ y^{(n)} $

Решением ДУ называется такая функция $y = y(x)$, которая при подстановке в уравнение обращает его в тождество. Это решение никогда не бывает единственным.

Задача о решении — это задача интегрирования ДУ.
График решения называется интегральной кривой данного уравнения.
Пример:

$ y'' = \cos(x) $ — это уравнение второго порядка $ y''=(y')' = \cos(x) $ $ y' = \int y'' dx = \int \cos(x) dx = \sin(x) + C $ $ y_{\text{o}} = \int (sin(x) + C) dx = \int \sin(x) dx + \int C dx = -\cos + C_1 + Cx $ — общее решение

Если обобщить пример, то в общем случае получим $ \phi(x, y, y', y'', ... y^{(n)}) = 0 $.
Решением такого уравнения является функция $ y = y(x, C_1, C_2, ... C_n ), C_i \in R $. При отсутствии других доп. условий, решением ДУ n-го порядка будет множество функций.

На графике это множество функций задает множество интегральных кривых (семейство интеральных кривых). Чтобы выделить одну кривую (решение), нужно задать доп. условие для определения этого решения - точка и направление. Эти условия называются краевыми (начальными). $$ y=y(x,C_1,C_2,C_n), C_i \in R $$

Общее решение ДУ

Частное решение получается из общего при подстановке конкретных значений $ C_1, C_2, ... C_n $, они находятся по заданным краевым условиям. Если заданы начальные условия, то говорят, что для уравнения задана задача Коши. Таким образом найти частное решение значит решить задачу Коши для данного уравнения.

Пример:

Дано: $ y'' = \cos(x), y(0) = 1, y'(0) = 2 $ Решение: $ y_{\text{o}} = -\cos + Cx + C_1 $ — общее решение Найдем частное решение: $ y'=sin(x) + C, y'(0) = 2 \Rightarrow 2 = \sin(0) + C \Rightarrow C = 2$ $ y = -\cos + Cx + C_1, y(0) = 1, C = 2 \Rightarrow 1 = -\cos(0) + C \cdot 0 + C_1 \Rightarrow C_1 = 2 $ $ y_{\text{ч}} = -cos(x) + 2x + 2 $ — частное решение

Дифференциальные уравнения первого порядка. Основные понятия, теорема о существовании и единственности решения

Основные понятия

Пусть есть $ y = f(x) $
$ \phi(x, y, y') = 0, $ — общий вид ДУ 1-го порядка
$ y' = g(x, y) $ — ДУ 1-го порядка, разрешенное относительно y'
$ y = y(x, C) $ — решение ДУ 1го порядка
$ \cdot(x_0, y_0) $ — начальное условие.

Геометрический смысл ДУ первого порядка

Вытекает из смысла производной. Пусть есть некое множество $D$, соответствующее обл. определ. функции. Такое множество точек с направлениями называется полем направлений. То есть ДУ первого порядка задает поле направлений для множества D. Решить такое уравнение — значит найти множество интегральных кривых, отвечающих заданному полю направлений.

О существовании и единственности решения

Рассмотрим уравнение $ y' = g(x, y) (1) $
Пусть $g$ и $g'$ непрервны на множестве $D$, тогда:

  • Тогда для любой точки этого множества существует решение уравнения (1), удовлетворяющее этой точке
  • Если есть два решения уравнения (1): $ y_1 = y_1(x), y_2 = y_2(x) $, и эти решения совпадают хотя бы в одной точке, т.е.
    $ \exists x_0 : y_1(x_0) = y_2(x_0) $
Тогда эти решения совпадают в одной точке, т.е. совпадают во всех точках $D$, т.е. решением является единственная интегральная кривая.

Основные виды ДУ

Неполные ДУ 1 порядка

  • Отсутствует $y$ $ y'=g(x) $ В этом случае уравнение решается непосредственным интегрированием: $ y = \int g(x) dx = G(x) + C $
  • Oтсутствует $x$ $ y'=g(y) $ Тогда по определению дифференциала: $ \frac{dy}{dx} = g(y) $ $ \frac{dx}{dy} = \frac 1 {g(x)} $ $ x = \int \frac{1}{g(y)}dy \Rightarrow x=\phi(y) + C $ Замечание: $ y'=g(y) $ $ \frac{dy}{dx} = g(y)$ — домножим на $dx$ $ dy = g(x, y)dx $

С разделяющимися переменными

Рассмотрим $ M(x) \cdot N(x) dx + P(x) \cdot Q(y) dy = 0 $
Данное уравнение надо преобразовать таким образом, чтобы перед дифференциалом $dx$ стояли функции, зависящии от $x$, а перед $dy$ — только от $y$.

Решение:
Задача привести к виду с разделенными переменными, по свойства интегралов $$ M(x) \cdot N(x) dx = -P(x) \cdot Q(y) dy $$ $$ \int M(x) \cdot N(x) dx = -\int P(x) \cdot Q(y) dy $$ Однородная функция: функция $z=f(x,y)$ называется однородной, если выполняются следующее условие $$ f(\alpha x,\alpha y) $$ Называется степень функции или порядок однородности любое однородное ду первого порядка может быть представлено к виду $ y' = g\left(\frac{y}{x}\right) $
